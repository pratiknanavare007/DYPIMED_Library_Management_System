How to run the Library Management System.. 

Technologies used PHP and MySQL

1.Download the zip file

2.Extract the file and copy librarymanagementsystem folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

	If database is not connected then change in the config file
	{
		host:(Change port)
		pass:(change pass)
	}
5.Create a database with name lms

6.Import lms.sql file(given inside the zip package in \Important File\Database File folder)

7.Run the script http://localhost/librarymanagementsystem
